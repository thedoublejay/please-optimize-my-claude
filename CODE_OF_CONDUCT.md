# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Enforcement

Violations may be reported to **<work@josejai.me>**. All reports will be reviewed and investigated promptly and fairly. The project maintainer is obligated to respect the privacy and security of the reporter.
